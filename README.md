# Focus Learning Log

A practical learning repo where GitHub contributions come from real progress.

## Areas
- `cs/` computer science notes and exercises
- `cybersec/` labs, commands, writeups
- `german-b2/` TELC B2 vocab, grammar, writing practice
- `labs/` mini-project experiments
- `scripts/` helper scripts
- `daily/` one file per day
- `templates/` reusable note templates

## Daily minimum
1. Add one useful artifact (note, exercise, lab result, or vocab).
2. Commit with a meaningful message.
3. Push.

## Commit examples
- `docs(german): add B2 vocab and sentence practice`
- `feat(cs): add binary search implementation`
- `notes(cybersec): summarize nmap service detection lab`
